HTTP Status Code Inspector - Web Security Project
نظرة عامة
مشروع تعليمي تفاعلي لشرح رموز حالة HTTP مع تحليل أمني شامل. المشروع يلبي متطلبات Project 19 لمادة أمن الويب.

المتطلبات المكتملة
متطلبات المشروع الأساسية:
✓ صفحة ويب بأزرار لرموز HTTP مختلفة

✓ كل زر يطلق طلب fetch (محاكاة)

✓ عرض رمز الحالة وشرح مفصل

✓ تضمين الآثار الأمنية لكل رمز

✓ جدول مرجعي للمطورين

متطلبات الدكتور الإضافية:
✓ مشروع JavaScript يحتوي على WebSocket (WS)

✓ فيديو شرح للمشروع

✓ ميزات إضافية للحصول على درجات أعلى

الميزات
ميزات أساسية:
محاكاة رموز HTTP: اختبار 16 رمز حالة مختلف

تحليل أمني: شرح المخاطر الأمنية لكل رمز

واجهة تفاعلية: ألوان حسب فئة الرمز (نجاح/خطأ/إعادة توجيه)

جدول مرجعي: جدول شامل مع تصنيف أمني

ميزات إضافية:
WebSocket آمن: اتصال wss:// مع شرح الأمان

سيرفر محلي: تشغيل عبر Python HTTP Server

نظام السجلات: تتبع جميع الطلبات والاستجابات

أداة لقطات داخلية: إمكانية التقاط الصور مباشرة من التطبيق

بحث وتصفية: البحث في الرموز والشرح

تحميل حركي: تحسين تجربة المستخدم

كيفية التشغيل
الطريقة 1: فتح الملف مباشرة
افتح ملف index.html في أي متصفح

الطريقة 2: تشغيل سيرفر محلي (مستحسن)
باستخدام Python:
python3 -m http.server 8000
ثم افتح في المتصفح: http://localhost:8000

الطريقة 3: باستخدام Node.js
إذا كان لديك Node.js:
npx serve .
أو:
npm install -g live-server
live-server
